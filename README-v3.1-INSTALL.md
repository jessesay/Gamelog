# GameLog v3.1 install notes

This patch adds the in-app guided setup system.

## Install

1. Copy everything in this patch into your GameLog project root.
2. Choose **Replace files** when Windows asks.
3. Commit in GitHub Desktop:

`Build GameLog v3.1 onboarding polish`

4. Push origin.
5. Redeploy Vercel.

## Test

- `/app?view=onboarding`
- `/start`
- App Home → Start setup
- Start setup → Apply starter taste
- Continue next step buttons
