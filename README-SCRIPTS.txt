GameLog Windows Scripts

Copy these three .bat files into the root of your GitHub-cloned GameLog folder, next to package.json:

- setup-gamelog.bat
- start-gamelog.bat
- update-gamelog.bat

Use them like this:

1. setup-gamelog.bat
   Run this once when setting up a fresh clone.

2. start-gamelog.bat
   Double-click this whenever you just want to start GameLog.

3. update-gamelog.bat
   Double-click this when you want to pull the newest GitHub version, install deps, approve pnpm builds, and start GameLog.

Do not commit .env.local.
